# DST
DST Shader Toy

## Build
install g++
```sudo apt install g++```
install glfw
```sudo apt install libglfw-dev```
download [glad](https://glad.dav1d.de/) (select gl Version 4.5 and only checkbox "Local Files")
then compile it
```g++ -c glad.c ```
locate glad headers at /usr/include
```./build.sh```

## Run
```sudo chmod +x ./bin/out```
```./bin/out```
